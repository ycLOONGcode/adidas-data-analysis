# Adidas虚拟电商数据深度洞察与可视化分析项目

## 项目概述
本项目基于和鲸社区50万条高仿真电商订单数据集，构建“数据清洗-多维度建模-可视化洞察-业务落地”的全链路分析框架，通过精细化数据挖掘揭示消费行为规律，为运动品牌电商精细化运营提供数据驱动决策支撑。

## 技术栈与方法论
- **数据工程**：Pandas 实现高效数据清洗（缺失值/重复值校验、异常值截断处理）与特征工程
- **可视化体系**：Pyecharts 构建地理空间热力图、时间序列趋势图、用户分层饼图等多维度可视化矩阵
- **分析模型**：集成 RFM 用户价值分层模型、消费行为归因分析、商品生命周期价值评估等方法论

## 核心分析模块
1. **时空消费洞察**：省份级消费热力分布映射区域市场潜力，小时/日级时间序列拆解下单规律
2. **商品价值分析**：全面分析电商销售数据，计算总GMV，客单价等总体指标，构建品类-子类-单品三级销售漏斗，定位核心盈利品类与TOP热销单品
3. **用户画像建模**：基于性别/年龄维度解析消费偏好，结合 RFM 模型实现8维用户分层（核心高价值用户/流失用户等）
4. **业务价值转化**：输出可落地的运营策略建议（商品备货、用户激活、区域营销资源分配）

## 产出物与应用场景
- 可交互 HTML 可视化图表（地区分布、销量趋势、用户分层等）
- 完整分析报告（含数据结论、模型解读、业务建议）
- 应用场景：电商数据分析教学案例、运动品牌运营决策参考、数据挖掘方法论实践模板
